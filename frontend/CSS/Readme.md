#  Tulika Lunkad – Portfolio Website (using HTML and CSS)

##  Overview

This portfolio website was developed as part of a frontend assignment to demonstrate:

- Semantic HTML structure  
- Advanced CSS styling  
- Responsive design principles  
- Modern UI/UX aesthetics  
- Clean project organization using Git  

The design follows a **dark AI-tech theme** with gradient backgrounds, glowing elements, smooth hover effects, and structured grid layouts.

---

## Tech Stack

- **HTML5** – Semantic structure  
- **CSS3** – Styling, animations, layout  
- **Google Fonts** – Playfair Display & Inter  
- **Font Awesome** – Icons  

---

## Key Features

### Design & UI
- Dark gradient AI-inspired background
- Glassmorphism card styling
- Gradient animated typography
- Glow effects and hover animations
- Circular profile image styling
- Elegant typography using Playfair Display

### Responsive Layout
- System-friendly design(works in any device)
- Adaptive grid system (2×2 layout for skills & projects)
- Optimized spacing and alignment

### Website Sections
- Hero Section with CTA buttons
- About Section with structured cards
- Skills Section (2×2 grid layout)
- Education Section
- Projects Section with preview cards
- Contact Form
- Footer
